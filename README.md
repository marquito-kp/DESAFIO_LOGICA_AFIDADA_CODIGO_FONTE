# DESAFIO LÓGICA AFIADA DE 21 DIAS

O desafio de lógica afiada de 21 dias é iniciativa do Código fonte TV, com o objetivo de manter constancia e prática em desafio de lógica aplicando JavaScript (JS)

## 1º Dia

No primeiro dia, o desafio foi manual, onde tivemos que redigir 3 algortimos, sendo:
* **Algortimo para acordar:** sendo atividades que realizo logo ao acordar;
* **Receita favorita:** passo a passo para realizar uma receita que gosto de cozinhar; e
* **Tarefas diárias:** listar as tarefas que realizo no meu dia a dia.

## 2º Dia

No segundo dia, já tivemos que listar quais dados encontravamos em um documento de identidade, por exemplo, e classifica-los.

Além disso, redigimos nosso primeiro código, que realiza o cálculo do IMC (Indice de Massa Corporal) que é cálculo através da divisão do Peso pela altura ao quadrado.

O arquivo está denominado "desafio04_ICMS.js".
